## Overview

A Responsive Visita Sevilla Clone built using only [HTML](https://html.spec.whatwg.org/), [CSS](https://www.w3.org/Style/CSS/specs.en.html) and [Tailwind CSS](https://tailwindcss.com/), following the [WCAG 2.2 guidelines](https://www.w3.org/TR/WCAG22/).

This project has been carried out for educational purposes.

## What have I learned?

* [Semantic HTML](https://developer.mozilla.org/en-US/docs/Glossary/Semantics#semantics_in_html) review.
* How to import font files in an [HTML](https://html.spec.whatwg.org/).
* How to apply the [WCAG 2.2 guidelines](https://www.w3.org/TR/WCAG22/) to a project.
* Using accessibility [HTML](https://html.spec.whatwg.org/) validators.
* Some good practices in [Responsive Design](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Responsive_Design) and Web layout in general.
